# 학습 키워드

상위 수준 애니메이션 API

- animateContentSize
- AnimatedVisibility

하위 수준 애니메이션 API

- animate*AsState: 단일 값에 애니메이션 적용
- updateTransition: 여러 값에 애니메이션 적용
- infiniteTransition: 무기한으로 값에 애니메이션 적용
- Animatable: 터치 동작으로 맞춤 애니메이션 빌드
